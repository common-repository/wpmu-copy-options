WPMU copy options

== Description ==

You can edit and clone any options of one blog and copy this items to the other blogs in multisite installation

== Installation ==

To use plugin, you will need:

* 	an installed and configured copy of WordPress version 3.5.x with multisite enabled
*	FTP or SFTP access to your web host

= New Installations =

1.	Download plugin archive in zip or gzipped tar format and
	extract the files on your computer. 

2.	Copy  plugin in the `wp-content/plugins`
	directory of your WordPress installation. Use an FTP or SFTP client to
	upload the contents of your archive to the new directory
	that you just created on your web host.

3.	Log in to the WordPress Dashboard and activate plugin.

4.  In Dmf Tools menu use 'WPMU copy options' item.

5.  Select source blog

6.  Press edit button to edit option value on as many options as you need and finally press "Save" button, press 'Confirm' button

7.  On confirm screen you can check needed options and blogs to apply changes to, press 'Save' button

8.  See result log
